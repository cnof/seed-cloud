## 架构图

## 项目介绍
提取通用功能，帮助项目快速开发
* 后端基于Spring Cloud Alibaba
* 注册中心、配置中心采用nacos
* 支持sa-token统一认证接入
* 脚手架同能齐全
* 支持可视化配置查询API接口
